# 麻将策略题库管理系统

一个用于展示和学习麻将策略题目的Web应用系统。

## 功能特点

1. **题目管理**：使用JSON格式管理题库数据
2. **题目展示**：上侧展示题目图片和原始链接
3. **答案解析**：下侧可点击显示答案图片和详细解析
4. **题目导航**：支持上一题/下一题切换
5. **类型筛选**：按题目类型筛选题目
6. **随机抽题**：随机抽取题目进行练习
7. **键盘支持**：支持键盘快捷键操作

## 文件结构

```
mahjong_question_bank/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # JavaScript交互逻辑
├── questions.json      # 题库数据
├── README.md           # 说明文档
└── img/                # 图片目录（可存放题目和答案图片）
```

## 题库数据格式

```json
[
  {
    "question_pic": "题目图片地址",
    "question_type": "题目类型",
    "question_urlori": "题目原始链接",
    "question_answer": "题目答案图片",
    "question_analyse": "题目解析"
  }
]
```

## 使用方法

1. 将题目图片和答案图片放入 `img/` 目录
2. 在 `questions.json` 中添加题目数据
3. 使用浏览器打开 `index.html`

## 键盘快捷键

- **← 左箭头**：上一题
- **→ 右箭头**：下一题
- **空格/回车**：显示/隐藏答案
- **R**：随机抽题

## 题目类型

系统支持三种题目类型：
1. **听牌判断** - 判断手牌的听牌情况
2. **和牌选择** - 选择和牌的最佳策略
3. **防守策略** - 制定防守策略避免放铳

## 开发说明

如需添加新题目：
1. 将题目图片和答案图片放入 `img/` 目录
2. 在 `questions.json` 中添加新的题目对象
3. 确保图片路径正确
